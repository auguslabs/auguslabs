## Icons (estructura unificada)

Carpeta base para todos los iconos publicos del sitio.
Usa minusculas y guiones en los nombres de archivo.

Estructura:
- `icons/tech/` → tecnologias (logos)
- `icons/solutions/` → iconos personalizados de servicios/soluciones
- `icons/ui/` → iconos generales de interfaz

### Tecnologias (`icons/tech/`)
Acepta `.svg` o `.png`:
- react
- astro
- tailwindcss
- typescript
- javascript
- firebase
- git
- github
- vite
- illustrator
- after-effects
- photoshop
- creative-cloud
- google-analytics
- search-console

Ejemplos:
